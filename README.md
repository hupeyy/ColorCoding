# Getting Started

## Prerequisites
- Node.js
- npm
- Python


## Installation
1. Clone the repo
```sh
git clone
```

2. Install NPM packages
```sh
cd frontend
npm install
```

3. Run the app
```sh
npm run dev
```

4. Open the app in your browser
```sh
http://localhost:5173
```

## Server Setup
1. Go to Backend folder
```sh
cd backend
```

2. Install NPM packages
```sh
npm install
```

3. Run the server
```sh
node index.js
```
